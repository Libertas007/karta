# karta

**karta** is a new tab page!

## Features

- an astronomical picture of a day
- a search bar
- a digital clock
- a time machine to browse past pictures of a day
- a current weather information determined by user location

## How to set up

1. Clone this repo
2. Grab an API key from [NASA](https://api.nasa.gov/) and [OpenWeatherMap](https://openweathermap.org/) and pass it to `.env`.
3. Install dependencies with your favourite package manager and run the server with Vite!
